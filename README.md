# 🚀 Flask + AWS Lambda Serverless Form Handler

This project demonstrates a simple serverless architecture where a Flask application sends form data to an AWS Lambda function via API Gateway.

---

## 🧠 Architecture

User → Flask App → API Gateway → AWS Lambda → Response

---

## ⚙️ Tech Stack

- Python
- Flask
- AWS Lambda
- API Gateway

---

## 📌 Features

- Simple HTML form (Name & Email)
- Flask handles frontend
- API Gateway exposes HTTP endpoint
- Lambda processes request and returns response

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
python app.py
